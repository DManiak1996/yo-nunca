---
description: "Identifica mejoras rápidas de UX/UI con alto impacto"
---

Activa el agente UX/UI Designer en modo "Quick Wins" para identificar mejoras rápidas de implementar pero con alto impacto en la experiencia de usuario.

**Objetivo:** Encontrar 5-10 mejoras que se puedan implementar en menos de 1-2 horas cada una pero que mejoren significativamente la UX.

**Instrucciones para el agente:**
1. Lee el agente definido en `.claude/agents/ux-designer.md`
2. Revisa rápidamente la aplicación buscando:
   - Mensajes de error poco claros
   - Falta de feedback visual
   - Problemas de contraste
   - Espaciados inconsistentes
   - CTAs poco visibles
   - Textos confusos
   - Falta de estados de loading
   - Validaciones faltantes
3. Prioriza por ratio impacto/esfuerzo
4. Proporciona código específico para cada mejora

**Entregables esperados:**
- Lista de 5-10 quick wins
- Para cada uno:
  - Problema actual (con screenshot textual o referencia al código)
  - Solución propuesta
  - Código específico para implementar
  - Impacto esperado
  - Esfuerzo estimado (minutos/horas)
