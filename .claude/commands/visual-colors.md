---
description: "Auditoría y propuesta de paleta de colores profesional"
---

Activa el agente de Diseño Visual de UI para realizar una revisión profunda de la paleta de colores y proponer una paleta profesional y coherente.

**Tipo de análisis:** Auditoría de paleta de colores

**Instrucciones:**
1. Lee el agente definido en `.claude/agents/visual-designer.md`
2. Analiza todos los colores utilizados en el proyecto actual
3. Evalúa coherencia cromática, armonías, contraste y accesibilidad
4. Identifica colores funcionales (éxito, error, warning, info)
5. Propón una paleta profesional con gradaciones (50-900)
6. Verifica compatibilidad con dark mode
7. Proporciona código específico para implementar la nueva paleta

**Entregables esperados:**
- Análisis de la paleta actual (qué está mal y por qué)
- Propuesta de paleta de colores profesional con:
  - Colores base (grays, primary, secondary)
  - Colores funcionales (success, warning, error, info)
  - Gradaciones completas (50-900)
  - Valores para dark mode
- Código de implementación (CSS variables, Tailwind config, o tokens)
- Verificación de contraste WCAG AA
- Referencias visuales de apps similares bien diseñadas
