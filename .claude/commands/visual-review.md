---
description: "Auditoría completa de diseño visual del proyecto"
---

Activa el agente de Diseño Visual de UI para realizar una revisión completa del diseño visual e identidad estética de este proyecto.

**Contexto del proyecto:**
- Analiza la identidad visual actual del proyecto
- Identifica la personalidad y emoción que transmite el diseño
- Evalúa la competencia visual si es relevante

**Tipo de análisis:** Auditoría completa de diseño visual

**Instrucciones:**
1. Lee el agente definido en `.claude/agents/visual-designer.md`
2. Revisa los archivos de estilos globales, componentes visuales y configuración de tema
3. Analiza paleta de colores, tipografía, espaciado, jerarquía visual y consistencia
4. Considera las skills disponibles (React, shadcn/ui, Next.js, Tailwind) para las propuestas
5. Proporciona un informe completo siguiendo el formato del agente

**Entregables esperados:**
- Resumen ejecutivo con puntuación de diseño visual
- Análisis detallado por áreas (color, tipografía, espaciado, jerarquía, etc.)
- Lista priorizada de mejoras (CRÍTICO/ALTO/MEDIO/BAJO) con código específico
- Sistema de diseño propuesto (tokens, paletas, escalas)
- 5-7 Quick Wins visuales
- Roadmap de implementación sugerido
