---
description: "Identifica mejoras visuales rápidas con alto impacto"
---

Activa el agente de Diseño Visual de UI para identificar mejoras visuales que se puedan implementar rápidamente pero con gran impacto en la percepción de calidad.

**Tipo de análisis:** Quick Wins visuales

**Instrucciones:**
1. Lee el agente definido en `.claude/agents/visual-designer.md`
2. Escanea rápidamente el proyecto en busca de inconsistencias visuales evidentes
3. Identifica mejoras que tomen menos de 1-2 horas cada una
4. Prioriza por impacto visual / esfuerzo de implementación
5. Proporciona código específico listo para copiar y pegar

**Entregables esperados:**
- Lista de 5-10 Quick Wins con:
  - Título descriptivo de la mejora
  - Problema específico (archivo, línea, componente)
  - Solución con código exacto
  - Tiempo estimado de implementación (minutos/horas)
  - Impacto esperado (bajo/medio/alto)
- Orden sugerido de implementación
- Código listo para implementar inmediatamente

**Ejemplos de Quick Wins:**
- Unificar border-radius inconsistentes
- Estandarizar sombras de componentes
- Corregir espaciado arbitrario a múltiplos de 4px
- Cambiar colores puros por paleta profesional
- Añadir estados hover faltantes en botones
- Mejorar contraste de texto de bajo contraste
